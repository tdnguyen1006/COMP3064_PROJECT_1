# Game Project 1 (Space Shooter)

This is a 2D space shooter game with 3 scenes â€“ Start, Main and Win. A player will control a spaceship to destroy or avoid meteors. A player will get points when destroying meteors or collecting coins. 5 points for collecting each coin and 1 point for each meteor. If a spaceship collides with a meteor, a player loses 1 life. After losing 3 lives, a player will lose.

Use W, A, S, D to move up, left, down, right
Use Spacebar to shoot
